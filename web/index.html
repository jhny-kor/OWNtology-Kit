온톨로지화 단계

선택 단계입니다. 꺼도 핵심 정제·검증은 항상 실행됩니다.

카카오톡

GitHub 스타 (선택)

API 키는 여기 입력하지 않습니다 — 환경변수(ANTHROPIC_API_KEY 등)로 설정하세요.

클라우드 동기화 (선택)

볼트를 원격 서버로 rsync push 합니다. SSH 키 인증이 미리 설정돼 있어야 하며 비밀번호는 저장하지 않습니다. 원격 MCP 이관은 deploy/README.md 참고.

비워두면 동기화 버튼이 비활성화됩니다. 방향은 로컬 → 원격(단방향, --delete 미러).

자동 수집된 인물 노트의 빈 필드를 채웁니다. 빈 필드가 있는 사람이 먼저 표시됩니다.
