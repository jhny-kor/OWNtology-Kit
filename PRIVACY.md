# 개인정보 안내 (PRIVACY)

owntology-kit은 **당신의 기기에서만** 동작하는 로컬 도구입니다. 수집한 데이터를
외부로 전송하지 않으며, 서버·클라우드로 업로드하지 않습니다. 그러나 다루는 데이터의
성격상 아래 사항을 반드시 이해하고 사용하세요.

## 무엇을 읽고, 어디에 저장하나

- **읽는 원본**: 카카오톡 로컬 DB(katok), iMessage/SMS `chat.db`(imsg), Mail.app 받은편지함,
  Apple 메모, Safari iCloud 탭 DB, (선택) 공개 GitHub 스타.
- **저장 위치**: `config.json`의 `vault_path`(기본 `~/Documents/my-owntology`). 모든 데이터는
  **평문 마크다운/JSON**으로 그 폴더에 저장됩니다. 암호화하지 않습니다.
- **외부 통신**: 링크 노드 생성 시 각 URL의 `<head>`(제목·og:description)를 가져오기 위해
  해당 사이트에 HTTP 요청을 보냅니다. GitHub 스타는 GitHub API를 호출합니다. LLM 요약
  (`--enrich`)을 켜면 스타 레포 메타데이터가 설정한 LLM 제공자(Anthropic/OpenAI)로 전송됩니다.
  그 외 개인 대화·메일·메시지는 어떤 외부로도 전송되지 않습니다.

## 제3자 개인정보

수집 데이터에는 **당신과 대화한 상대방의 개인정보**(이름·전화번호·메시지 내용)가 포함됩니다.
1:1 카카오 대화 상대는 자동으로 `people/` 인물 노트(스텁)로 만들어집니다. 이 데이터를
본인 외의 사람과 공유하거나 공개하는 것은 상대방의 프라이버시를 침해할 수 있으며, 지역에
따라 법적 책임이 따를 수 있습니다. **볼트는 개인용으로만 사용하세요.**

## 안전 수칙

- 볼트를 **공개 저장소(GitHub 등)나 공유 폴더에 두지 마세요.** iCloud Drive/Dropbox/
  Google Drive 동기화 경로도 피하세요(`kit.py doctor`가 경고합니다).
- 웹 설정 화면(`kit.py web`)은 인증이 없어 **127.0.0.1(localhost)에만** 바인딩됩니다.
  외부에 노출하지 마세요.
- 민감 노트는 `sensitivity: sensitive/secret` frontmatter로 기본 차단되고, 반환 텍스트의
  비밀번호·API 키·인증번호는 마스킹됩니다(`kitlib/security.py`). 그래도 원본은 평문으로
  볼트에 남습니다.
- 더 이상 필요 없으면 볼트 폴더를 직접 삭제하면 모든 수집 데이터가 제거됩니다.

## 통제권

- 어떤 소스를 수집할지는 전적으로 당신이 켭니다(기본은 전부 꺼짐).
- 자동 추출된 개인계층 후보(결정·사건·선호)는 전부 `proposed` 상태이며, 당신이 승인하기
  전에는 확정 노트가 되지 않습니다.
